# Bluestock Mutual Fund Analytics Capstone

## Overview

This project develops an end-to-end Mutual Fund Analytics Platform using Python, SQL, and Power BI.

## Objectives

* Data Ingestion & ETL
* Data Cleaning
* SQL Database Design
* Exploratory Data Analysis
* Risk & Performance Analytics
* Dashboard Development
* Advanced Analytics

## Datasets

1. Fund Master
2. NAV History
3. AUM by Fund House
4. SIP Inflows
5. Category Inflows
6. Industry Folio Count
7. Scheme Performance
8. Investor Transactions
9. Portfolio Holdings
10. Benchmark Indices

## Technologies

* Python
* Pandas
* NumPy
* SQLAlchemy
* SQLite
* Matplotlib
* Seaborn
* Power BI
